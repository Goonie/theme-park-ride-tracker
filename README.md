# Theme Park Ride Tracker

Web app to keep track of rides taken during theme park visits

## Installation

- Clone the repository and run `pnpm install` to install its dependencies;
- Install [Supabase CLI](https://supabase.com/docs/guides/local-development/cli/getting-started?queryGroups=platform&platform=npm) and start the development server by running `supabase start`;
- Edit or copy `.env` to `.env.local` to add the local development server's key and URL;
- Finally, run `pnpm dev` to run the web application locally.
